# imooc-vue

> A Vue.js project

## Build Setup
1. 安装node.js到全局
2. 本地部署vue.js项目
``` bash
# install dependencies 安装依赖 版本号建议@3
npm install @3

# serve with hot reload at localhost:8081
npm run dev
```
